# Skillect | AI-Powered Career Intelligence

Skillect is a premium, AI-driven platform specifically designed for NIET students to bridge the gap between academia and industry. Built with a "GenZ-first" design philosophy, it combines high-end aesthetics with powerful AI tools.

## 🚀 Core Features

- **Smart Mentoring (MentorMatch)**: AI-driven discovery of faculty members matching your research and career goals.
- **AI Skill Audit**: Deep-scan of your current arsenal against industry-standard requirements.
- **Dynamic Roadmaps**: Interactive 4-week winding path roadmaps personalized by AI.
- **ATS Intelligence**: High-fidelity resume analysis with strategic refinement suggestions and readiness scores.
- **Skillect Resume Builder**: ATS-optimized resume generator powered by LLaMA 3.3.
- **Anonymous Doubt Box**: A safe space for students to ask questions and get instant AI-verified answers.
- **Domain Pulse**: Real-time global tech news and internship alerts.

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: TailwindCSS, Framer Motion (Animations), Lucide React (Icons)
- **Intelligence**: Groq LLaMA 3.3-70B API
- **Auth/Backend**: Supabase
- **Visuals**: DiceBear Avatars API

## 🎨 Design System

- **Typography**: Inter (Weights 400-900)
- **Corners**: 2.5rem - 3rem (Premium Softness)
- **Palette**: NIET Crimson (#E31E24) & Slate Dark (#1e293b)
- **Effects**: Glassmorphism, 3D Elevation, Dynamic Gradients
